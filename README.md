# Email Request Alert – Power Automate Flow

This project demonstrates a simple but practical automation built in Microsoft Power Automate.  
The flow monitors incoming emails and triggers an alert when the subject contains a specific keyword (“Request”).

---

## 🔧 Features

- Detects incoming emails with a defined keyword  
- Uses conditional logic to evaluate the subject  
- Sends an automated alert message  
- Clean and minimal structure  
- Suitable for enterprise environments (e.g., Siemens Energy)

---

## 🧠 How It Works

1. **Trigger:**  
   The flow starts when a new email arrives.

2. **Condition:**  
   The subject line is checked for the keyword **“Request”**.

3. **Action (True branch):**  
   Sends an alert message with structured details.

4. **Action (False branch):**  
   No action.

---

## 📬 Alert Message Format

[ALERT] New request detected

Subject: <email subject>
From: <sender>
Received: <timestamp>

This is an automated notification generated by Power Automate.
